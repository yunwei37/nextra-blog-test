# Welcome to Our Blog

This is an example blog page written in **MDX**. You can use all the power of Markdown with React components mixed in seamlessly.

## What makes MDX special?

MDX allows you to:

- Write content in Markdown
- Use React components anywhere
- Import JavaScript modules
- Create interactive documentation

## Code Examples

You can include syntax-highlighted code blocks:

```javascript
function createBlogPost(title, content) {
  return {
    title,
    content,
    publishedAt: new Date(),
    slug: title.toLowerCase().replace(/\s+/g, '-')
  }
}

const post = createBlogPost('My First Post', 'Hello, world!')
console.log(post)
```
