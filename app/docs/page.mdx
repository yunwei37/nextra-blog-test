# Documentation

Welcome to the documentation section of this Nextra site.

## What is Nextra?

Nextra is a framework on top of Next.js, that lets you build content focused websites. It has all the great features from Next.js, plus extra power to create Markdown-based content with ease.

## Key Features

### 📝 Markdown Support
Write content in Markdown and MDX with full React component support.

### 🎨 Built-in Themes
Choose from beautiful, customizable themes like Docs and Blog themes.

### 🔍 Full-text Search
Built-in search functionality powered by Flexsearch.

### 📱 Mobile Optimized
Responsive design that works perfectly on all devices.

### ⚡ Performance
Built on Next.js for optimal performance and SEO.

## Getting Started

To get started with your Nextra site:

1. Install the dependencies
2. Configure your `next.config.mjs`
3. Create your layout
4. Start writing content in MDX

Check out the [official documentation](http://nextra.site/docs) for more details.
