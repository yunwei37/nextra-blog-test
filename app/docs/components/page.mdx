## Custom Components

You can also create and use your own React components in MDX:

```jsx
// components/MyComponent.jsx
export function MyComponent({ children }) {
  return (
    <div className="my-custom-component">
      {children}
    </div>
  )
}
```

Then use it in your MDX:

```mdx
import { MyComponent } from '../components/MyComponent'

<MyComponent>
  This is my custom component content!
</MyComponent>
```
