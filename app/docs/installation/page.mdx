# Installation

Learn how to install and set up Nextra for your project.

## Prerequisites

Before you begin, make sure you have:

- Node.js 18.17 or later
- A package manager (npm, pnpm, yarn, or bun)

## Quick Start

### 1. Install Dependencies

```bash
npm i next react react-dom nextra nextra-theme-docs
```

### 2. Add Scripts

Add these scripts to your `package.json`:

```json
{
  "scripts": {
    "dev": "next",
    "build": "next build",
    "start": "next start"
  }
}
```

### 3. Configure Next.js

Create a `next.config.mjs` file:

```js
import nextra from 'nextra'
 
const withNextra = nextra({
  // Nextra configuration
})
 
export default withNextra({
  // Next.js configuration
})
```

### 4. Add MDX Components

Create `mdx-components.js`:

```js
import { useMDXComponents as getComponents } from 'nextra-theme-docs'
 
export const useMDXComponents = getComponents
```

### 5. Create Layout

Set up your root layout in `app/layout.jsx` with the Nextra theme components.

## Development

Start the development server:

```bash
npm run dev
```

Your Nextra site will be available at `http://localhost:3000`.
