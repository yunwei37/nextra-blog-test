# About

This is a sample Nextra documentation site created following the [official Nextra docs theme guide](http://nextra.site/docs/docs-theme/start).

## What's Included

This project demonstrates:

- ✅ Nextra Docs Theme setup
- ✅ MDX content with React components
- ✅ Navigation and page structure
- ✅ Responsive design
- ✅ Search functionality (when configured)
- ✅ Code syntax highlighting

## Project Structure

```
├── app/
│   ├── layout.jsx          # Root layout with Nextra theme
│   ├── page.mdx           # Homepage
│   ├── _meta.js           # Navigation configuration
│   ├── docs/              # Documentation section
│   │   ├── page.mdx
│   │   ├── _meta.js
│   │   └── installation/
│   │       └── page.mdx
│   └── about/
│       └── page.mdx       # This page
├── next.config.mjs        # Next.js + Nextra configuration
├── mdx-components.js      # MDX components configuration
└── package.json           # Dependencies and scripts
```
